メソッドのSQLの挙動を確認する

to_sqlメソッドを使う

$ bin/rails c
[1] pry(main)> user = User.first
[2] pry(main)> user.like_posts.to_sql
=> "SELECT `posts`.* FROM `posts` INNER JOIN `likes` ON `posts`.`id` = `likes`.`post_id` WHERE `likes`.`user_id` = 1"

整理すると

SELECT
  *
FROM
  posts
INNER JOIN
  likes
ON
  post.id = likes.post_id
WHERE
  likes.user_id = 1
;

というSQLが発行されていることがわかる。
